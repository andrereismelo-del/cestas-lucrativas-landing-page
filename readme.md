# Cestas Lucrativas – Dia dos Namorados

Landing page de vendas para o guia digital "Cestas Lucrativas", focado em ensinar como montar e vender cestas personalizadas para o Dia dos Namorados.

## Tecnologias

- HTML/CSS/JavaScript puro (sem dependências ou frameworks)
- Google Fonts (Playfair Display + DM Sans)
- Hospedagem: Netlify (site estático)

## Como rodar localmente

Basta abrir o arquivo `index.html` no navegador, ou usar qualquer servidor HTTP estático:

```bash
npx serve .
```
